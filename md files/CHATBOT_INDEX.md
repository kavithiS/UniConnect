## 🚀 Getting Started

### For Users
1. Open the app: `http://localhost:5183`
2. Login with your credentials
3. Look for the **purple chat button** (bottom-right)
4. Click to open the chatbot
5. Type your question (e.g., "Show me groups")

### For Developers
1. Read: `CHATBOT_CODE_REFERENCE.md`
2. Review: Backend changes in `aiChatController.js`
3. Review: Frontend changes in `ChatBot.jsx`
4. Build: `npm run build` (frontend)
5. Customize: Update response templates as needed

### For DevOps
1. Review: `CHATBOT_CHANGES_LOG.md`
2. Verify: Frontend build (1890 modules, 0 errors)
3. Check: Backend running on port 5008
4. Deploy: Follow deployment checklist
5. Monitor: Check logs for errors
